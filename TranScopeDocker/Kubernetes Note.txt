1.Sign In Docker Hub (https://hub.docker.com/)
	docker login
2.Build ulang image api_transcope
	pakai build hanya untuk create image
	docker compose -f docker-compose.API_TranScope.yml build
	akan terbentuk image api_transcope sesuai dengan docker file nya
3.Re-Tag and Push image api_transcope into Docker hub
	docker tag api_transcope dockerguntech/api_transcope
	docker push dockerguntech/api_transcope
4.running kubernetes.API_TranScope
	kubectl apply -f kubernetes.API_TranScope.yml
5.Check deployment
	kubectl describe deployment api-transcope
	kubectl get deployment api-transcope -o yaml

6.Check Progress of deployment
	kubectl get pods
	kubectl top pod api-transcope (enable metrix server terlebih dahulu)
7.Check Services
	kubectl describe service api-transcope-service
	kubectl get services (lihat load balancer ip public nya)
	http://localhost:5000/swagger/index.html
8.Drop Deployment
	kubectl delete -f kubernetes.API_TranScope.yml
	atau
	kubectl delete deployment api-transcope
	kubectl delete service api-transcope
